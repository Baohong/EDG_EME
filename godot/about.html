Background

Creating metadata has been a challenge for agencies across the US for many years, often due to lack of tools that provide a simple editing interface that meets all requirements. To address this challenge, the EPA developed the EPA Metadata Editor (EME) in 2006. Its streamlined design, integrated help system, and use of defaults and drop-downs simplify the process of creating metadata that meets FGDC requirements.

The EME is compatible with ArcGIS 10 and can be used either as an ArcCatalog extension or a standalone tool. Key features of the EME include a customizable synchronizer and an EPA validation tool.

EME Technology Details
    EME Installation Prerequisites:
  • Microsoft .NET Framework 3.5
  • ArcGIS 10* (or higher)
  • FGDC Patch for ArcGIS*
    *Only required for EME as an ArcGIS extension
    EME Technology Stack:
  • Microsoft Visual Studio 2010 (VB.NET)
  • Microsoft .NET framework 3.5
  • ArcCatalog 10
  • Microsoft Access
